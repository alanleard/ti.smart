Parameter Definitions
id The primary API required id
alt_id The second required id (example: hub & device)
body JSON formatted body to send with a POST
type The type of API call (GET,POST,PUT,DELETE).  Currently can only be used to override GET.
onload The onload callback function.
onerror The onerror callback function.
